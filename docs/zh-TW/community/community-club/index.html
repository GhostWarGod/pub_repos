俱樂部

Crypto Arena俱樂部運營計劃​

目標​

Crypto Arena致力於構建全球領先的開放式區塊鏈休閒競技生態平台，通過去中心化俱樂部運營模式賦能社區，打造開放、創新、共贏的遊戲生態系統，為所有參與者創造可持續價值。

加入條件​

  1. 有成熟的社區用戶群、或者有上線的區塊鏈項目。
  2. 具備發展和運營社區的能力。

俱樂部權益​

1. 基礎權益​

  1. 俱樂部享受平台內消費金額30%的佣金，如購買道具等。

  2. 俱樂部享受遊戲門票收入的10+%。

  3. 支持無限層級發展模式，發展的用戶均為本俱樂部成員。

    • (1)俱樂部創建者擁有推廣獎勵的頂層控制權，按需給直推的成員分配佣金比例。
    • (2)每一級成員可以給直推的成員調整佣金比例，調整的幅度為0.1%。
    • (3)每個加入俱樂部的成員，默認享受5%的推薦佣金。非俱樂部成員的佣金為3%。

  4. 自動綁定俱樂部。所有俱樂部成員通過自己的邀請連結邀請好友，通過連結進入的平台的用戶自動加入本俱樂部，不能退出。

  5. 運營數據分析報告。

  6. 專屬平台商務1對1服務。

  7. 官方媒體支持。

  8. 7×24小時技術支持。

定制權益​

  1. 擁有1~3個項目代幣的免費上架額度，上架的代幣可以作為門票使用。超出額度的上架費用為15000 USDT/項目。

  2. 自定義個性化的遊戲場，包括門票，地圖，音樂等

  3. 自定義個性化的大廳主題(需技術團隊協助定制)

  4. 自定義遊戲地圖(需技術團隊協助定制)

  5. 支持自有項目加入遊戲生態，平台給與全方位合作支持。

遊戲矩陣支持​

俱樂部權益適用平台所有產品，無額外費用，永久有效。計劃首批上線的8款經典遊戲：

  • Crypto Snake（貪吃蛇）
  • Crypto Tetris（俄羅斯方塊）
  • Crypto Bomberman（炸彈人）
  • Crypto Bird（像素鳥）
  • Crypto Candy Crush（消消樂）
  • Crypto Fishing（大魚吃小魚）
  • Crypto Tank（坦克大戰）
  • Crypto Mario World（馬里奧）
