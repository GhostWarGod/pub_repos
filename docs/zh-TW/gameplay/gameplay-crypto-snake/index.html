Crypto Snake

CryptoSnake是一款經典的多人線上對戰遊戲，基於傳統的貪吃蛇玩法加入了多人競爭元素、即時競技和區塊鏈等特性，玩家控制一條蛇，透過吃食物讓蛇變長，並盡量避免碰到其他蛇或牆壁。遊戲的目標是盡量讓自己的蛇變長，擊殺其他玩家，成為地圖上最強大的蛇。

1. 基本操作​

控制方向： 玩家透過滑動螢幕來控制蛇的方向。蛇的頭部指向滑動的方向，尾部會跟隨。
加速： 在遊戲中，玩家可以透過點擊螢幕的某個地方來讓蛇進行短時間加速。加速時蛇會消耗一部分體長，但會迅速增加移動速度。
吃食物： 地圖上會隨機生成食物，玩家的蛇碰到食物後會變長，體積也會增大。
使用道具： 使用道具會產生相應的遊戲體驗。

2. 蛇的成長​

  1. 吃食物： 當蛇吃到食物時，蛇的體長增加，蛇的得分也會相應增加。
  2. 蛇的體長增長有上限。
  3. 碰撞與死亡： 如果你的蛇頭碰到其他玩家的蛇身體，或者撞牆了，遊戲就會結束。玩家的蛇會死亡，變成食物供其他玩家吃。

3. 多人對戰​

  • 多人競技： 遊戲是一個即時線上多人競技場，玩家與全球的其他玩家同時競技。在同一個地圖中，所有玩家的蛇都在爭奪食物，隨時可能與其他蛇發生碰撞。
  • 擊殺排行榜： 遊戲有排名系統，玩家可以透過吃食物、避免被撞擊逐步成長起來，在成長的過程中擊殺其他的玩家，爭取成為地圖上擊殺數量最多的蛇，最終登上排行榜。

4. 遊戲策略​

  • 圍繞敵人： 在遊戲中，玩家可以透過策略包圍其他小蛇，利用對方的失誤讓其撞牆或自己撞到你的蛇身。特別是當敵人的蛇頭接近時，適時進行加速，可以製造更多機會。
  • 利用食物成長： 在遊戲中，盡量多吃食物，尤其是當你發現其他玩家的蛇死亡時，可以迅速趁機吃掉大塊的食物，迅速成長。
  • 避免衝動： 不要盲目追求速度和加速，有時候保持穩定和冷靜的玩法更有優勢，尤其是避免與其他玩家發生不必要的碰撞。

5. 房間模式​

CryptoSnake通常提供多種不同的房間模式，主要包括：

  • 自由場： 新手玩家的學習模式，練習遊戲的基本操作。
  • 無盡模式： 系統房間，玩家需要支付門票進入遊戲，在公平規則的情況下互相競爭，擊殺對手，獲得收益，看誰能成為地圖上最強的蛇。蛇的體長增長有上限。
  • 俱樂部模式： 俱樂部可以自訂房間，使用自己的數位資產作為門票。
  • 自訂房間： 玩家自訂規則，需要支付門票進入遊戲。

6. 遊戲道具與皮膚​

  • 皮膚NFT： 玩家透過USDT購買不同的蛇皮膚，個性化自己的蛇形象。
  • 道具： 部分版本的遊戲中可能會提供特定的道具（如保護罩、加速道具等），玩家可根據需要使用這些道具來提升遊戲體驗。

7. 遊戲結束條件​

  • 碰撞規則：
  1. 撞到其他蛇的身體會死亡。
  2. 兩蛇頭部相撞，身體短的蛇死亡。
  3. 兩蛇頭部相撞，身體長度相等，後進入遊戲的蛇死亡。
  • 退出： 玩家可以關閉瀏覽器，退出遊戲，門票不予退回

8. 結算​

  • 門票： 自由場免費，其他房間模式需要購買門票入場開始遊戲。系統使用主流數位資產購買門票，也支援社群項目的數位資產購買門票。
  • 擊殺獎勵： 擊殺發生後立即戰鬥結算，被擊殺的玩家支付的門票作為獎勵。
  • 被擊殺獎勵： 獲得一定數量的鑽石獎勵
